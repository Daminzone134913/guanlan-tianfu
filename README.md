# 观澜 · 东方天赋定位

> 水无常形，天赋无定式。聊一聊，看见你没发现的天赋。

## 关于观澜

观澜是一套**东方哲学风格的天赋定位系统**。不同于西方的问卷式测评（MBTI/盖洛普），观澜用**对话式聊天**的方式，在轻松的交流中帮你发现自己的天赋。

## 三个版本

| 版本 | 时长 | 题数 | 产出 |
|------|------|------|------|
| [快速版](./quick/) | 5-7分钟 | 5题 | 天赋标签 + 分享卡片 |
| [专业版](./pro/) | 约21分钟 | 15题 | 完整报告 + 百日行动计划 |
| [深度版](./deep/) | 81分钟 | 21题 | 深度天赋画像（可分次完成）|

## 天赋原型系统

观澜以水的十种形态命名天赋原型：

| 原型 | 定义 | 金句 |
|------|------|------|
| 🌊 涌泉 | 灵感不竭，万物皆可创 | 泉从心出，何须外求 |
| 🌊 听潮 | 于无声处听惊雷 | 潮汐有信，智者先闻 |
| 🌊 破壁 | 规则是拿来打破的 | 不破不立，我即破局者 |
| 🌊 深潭 | 深水无声，静照万物 | 潭深水静，一念通透 |
| 🌊 飞瀑 | 一泻千里，震撼人心 | 我生来，就是要被看见的 |
| 🌊 长河 | 志在千里之流 | 大河不言，自有方向 |
| 🌊 云雾 | 身在局中，眼在局外 | 拨云见日，我即是那朵云 |
| 🌊 溪行 | 遇山开路，遇石绕行 | 水无常形，我自成流 |
| 🌊 巨浪 | 一浪既起，万物相随 | 不做跟随者，我做浪潮本身 |
| 🌊 静流 | 表面波澜不惊，底下自有乾坤 | 不争，故天下莫能与之争 |

## 技术特点

- 纯前端单文件 HTML，零依赖
- 东方美学 UI（深蓝紫渐变 + 金色点缀 + 宋体标题）
- Web Speech API 语音输入（Chrome/Safari）
- Canvas 生成天赋标签分享卡片
- 支持 PWA（可添加到手机桌面）
- 微信内友好体验（降级方案 + 引导）
