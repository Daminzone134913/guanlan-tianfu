## 🌊 项目简介

「观澜」是一个完全自主开发的AI天赋测评系统，融合东方哲学智慧与现代心理学理论，通过21道精心设计的问题，帮助用户发现自身天赋与优势。

**设计理念：**
- 🎯 **对标国际**：学习Dan Koe天赋测评体系，但更贴合东方人思维
- 🗣️ **语音交互**：支持语音输入，像聊天一样轻松完成测评
- 📊 **深度报告**：9大板块详实分析，优于市面同类产品
- 🆓 **完全免费**：无需付费，完成即得完整报告

## ✨ 核心特性

### 🎤 语音输入
- 点击麦克风按钮即可开始录音
- 实时语音识别，自动转换为文字
- 识别完成后自动发送

### 📝 21道深度问题
分为5个阶段，层层递进：
1. **暖场·入水** - 建立信任，放松心态
2. **观流** - 观察日常状态与心流体验
3. **溯源** - 追溯成长经历与关键事件
4. **观照** - 反思内在模式与价值取向
5. **定位** - 明确天赋方向与行动路径

### 📊 9大板块报告
- 天赋素描
- 核心天赋一句话
- 天赋能量图谱
- 成长轨迹回溯
- 潜在盲区提示
- 适合的发展路径
- 行动建议
- 资源推荐
- 长期成长指引

### ⏱️ 81分钟从容节奏
取"九九八十一难/九九归一"之意，不催促，让用户从容思考。

## 🚀 技术栈

- **纯前端实现**：单个HTML文件，零依赖
- **Web Speech API**：浏览器原生语音识别
- **响应式设计**：适配手机/平板/电脑
- **东方美学UI**：深色渐变 + 金色点缀

## 📦 本地运行

1. 克隆仓库
```bash
git clone https://github.com/Daminzone134913/guanlan-tianfu.git
cd guanlan-tianfu
```

2. 直接用浏览器打开 `index.html`

或使用本地服务器：
```bash
# Python
python3 -m http.server 8000

# Node.js
npx serve
```

然后访问 `http://localhost:8000`
